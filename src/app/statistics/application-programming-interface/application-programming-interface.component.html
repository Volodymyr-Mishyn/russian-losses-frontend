API / Data access

/api/mod

Endpoint for accessing the data from the Ministry of Defense of Ukraine.

  • start: string in Date ISO format

    end: string in Date ISO format

  • flat: boolean (optional)

Examples:

{{ baseUrl }}/api/mod?start=2022-03-24&end=2022-03-26 

{{modData|json}}

{{ baseUrl }}/api/mod?start=2022-03-24&end=2022-03-26&flat=true 

{{modDataFlat|json}}

/api/oryx

Endpoint for accessing the data from the Oryx OSINT.

country: string "russia"| "Ukraine"

Example:

{{ baseUrl }}/api/oryx?country=russia 

{{oryxData|json}}
